PORTFOLIO SITE
PORTFOLIO SITE: Timothy Hsu's Portfolio
HTML  CSS  JavaScript
The Plan
The portfolio site is created to be a place to share my work and for people to find out more about me. I want people to not only find out about the projects that I have done but also the thought process and design thinking in arriving at the solutions. The site structure and design will reflect how I think and reflect myself as an individual.
The Process
Each of the projects will be described with the tools used and the thought process on how I created the websites or interfaces. Screenshots and mockups will portray how the website will look like on desktop and mobile formats. I want to create a portfolio site with a clear organization scheme with ease of use and functionality in mind. Since this site originated from all my ideas, I went ahead and build the website without doing any mockups or wireframes. I had the idea in mind and started coding and made changes along the way.
Design
The design of the site all came with idea of using the circle shape and creating the elements based on it. I wanted to use the circle in my layout as we donâ€™t often see it being used as much in design when compared to grids and squares. I decided to put my name in the circle and used vertical text for the navigation and elements to create a distinct look. A color scheme of blue, green, and light orange was chosen to create the site. The overall feel of the portfolio site was planned to be a simple layout with full screen background, grid layout for the projects, and with vivid colors for the elements.
