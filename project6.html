REACT API: News and Movie API
React.js  HTML   CSS  
Using React, we created two applications to demonstrate requesting and fetching data using API. With the News API, we can search for a term and find the related news displayed in a tabular format. I created the News app with a easy to use interface using clean tabular layout and vibrant colors to show the news in a pleasing manner. Similarly, with the Movies API, we can filter for a genre and find all related movies listed in that genre. I decided to use a darker theme with black and white colors to simulate a movie theatre experience.
