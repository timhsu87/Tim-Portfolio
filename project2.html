LANDING PAGE: Personal Portfolio Site
HTML  CSS   Balsamiq
The Plan
The goal of my landing page is to be a hub where people can see samples of my projects, find out more about me, and see my design style. I love the clean, simple design approach and find that with this approach, it is easier for users to find what they are looking for in a visually attractive way. I decided to incorporate images that will convey creativity and design.
The Process
I started the design with creating wireframes in Balsamiq to get a rough layout of how my site will look like. Since this was my personal site, it didnâ€™t take long for me to settle on the design and layout of the wireframe. I wanted to use HTML and CSS to create a visually appealing site that best represent my work and design style.

Design
With the aspiration to create interfaces that are both functional and beautiful, I decided to use a grid layout with design related images to add to the visual experience. With a clean simple layout using white space, font choice, and a repetitive structure, with the hope that all these elements would portray my design style effectively. The full screen video background was used to add to the overall visual experience of the site.
